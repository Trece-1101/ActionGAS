# ActionGAS

ActionGAS es un prototipo de juego de acción desarrollado en Unreal Engine 5 con C++. El proyecto está centrado en la implementación profunda del Gameplay Ability System (GAS), gestionando la ejecución de habilidades, la replicación de atributos, Gameplay Effects y el manejo de inputs para sistemas de combate escalables y modulares.
***

***Engine:*** Unreal v5.7.3

***Lenguaje:*** C++ / Blueprints

***Plataforma:*** PC

***Genero:*** Acción / Prototipo Técnico
